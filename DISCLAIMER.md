## Unofficial API

This app talks to xBloom's cloud API using request formats that were
reverse-engineered by the community (see credits in the README). This API is
private and undocumented:

  your account, your machine, or your coffee.

## Your credentials

When you self-host this app, **your xBloom email and password are used by your
own deployment only** — they are read from environment variables on your own
server and are never sent anywhere except xBloom's own API to log in. They are
not stored in the database and not shared with the project authors.

Never commit your credentials to git. Never deploy this app publicly without
protecting it (see the README's deployment section).

## Brewing recipes

Recipes are generated by an AI model and are a starting point, not
professional advice. Always sanity-check temperatures and parameters before
brewing. Hot water is hot. Use common sense.
